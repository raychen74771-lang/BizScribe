### 🌍 简介 | Introduction | 소개

**[中文]**
恩流 (FlowScribe) 是一款专为全球教会文字事工打造的“零门槛、低成本”桌面软件。它能全自动将讲道录音转化为排版精美、神学词汇准确的讲章文字。无需懂代码，普通办公电脑即可直接运行。

**[English]**
FlowScribe is a zero-barrier, low-cost desktop application designed specifically for global church textual ministries. It automatically converts sermon audio into beautifully formatted text with highly accurate theological terminology. No coding skills required; it runs smoothly on standard office laptops.

**[한국어]**
FlowScribe(은류)는 전 세계 교회 문서 사역을 위해 특별히 제작된 "진입 장벽이 없는, 저비용" 데스크톱 프로그램입니다. 설교 녹음을 신학적 용어가 정확하고 깔끔하게 편집된 설교문으로 자동 변환해 줍니다. 코딩을 몰라도 일반 사무용 컴퓨터에서 쉽게 사용할 수 있습니다.

### 💖 我们的初心 | Why we built this | 제작 배경

**[中文]** 每到周末，全球有无数充满恩典的讲道被宣讲。但要把这些宝贵的信息沉淀为文字，往往需要文字事工的同工们熬夜听打、反复校对。讲台上的信息是活水，我们希望这水能流得更远。通过这款工具，我们希望把同工们从繁重的“听打”中解放出来，让大家把宝贵的时间花在牧养上。

**[English]** Sermons are living water, and we want them to flow further. We built FlowScribe to free coworkers from the exhausting task of manual transcription, allowing them to dedicate more time to deep editing and pastoral care.

**[한국어]** 설교는 생수이며, 우리는 그 생수가 더 멀리 흘러가길 원합니다. 문서 사역자들을 밤샘 타이핑 작업에서 해방시켜, 목양과 영혼 돌봄에 더 많은 시간을 쏟을 수 있도록 이 프로그램을 만들었습니다.

### ✨ 核心特色 | Core Features | 핵심 기능

**1. 🧠 双脑协作机制 | "Dual-Brain" Engine | '두뇌 협업' 엔진**
* **ZH**: 左脑极速听写（离线不耗流），右脑（DeepSeek V3）懂神学上下文，自动纠错并按段落排版。
* **EN**: Left-brain handles blazing-fast offline transcription, while the right-brain (DeepSeek V3) fixes theological typos, removes filler words, and formats paragraphs.
* **KR**: 좌뇌는 오프라인 환경에서도 빠른 음성 인식을 수행하고, 우뇌(DeepSeek V3)는 신학적 문맥을 파악하여 오타를 수정하고 단락을 깔끔하게 정리합니다.

**2. 🎙️ 讲员风格克隆 | Style Cloning | 설교자 스타일 복제**
* **ZH**: 上传讲员过去的旧讲稿，软件就能学习并模仿这位讲员的语气和断句习惯。
* **EN**: Upload past sermon transcripts to clone the speaker's unique tone and phrasing habits.
* **KR**: 과거 설교 원고를 업로드하면 AI가 설교자의 어조와 문장 나누는 습관을 학습하고 모방합니다.

**3. 💻 零内存切片（老电脑福音） | Zero-RAM Slicing | 제로 메모리 (저사양 PC 지원)**
* **ZH**: 独创物理切片黑科技，3个小时的超长讲道在老旧办公本上也能流畅跑完，永不卡死崩溃。
* **EN**: Innovative physical audio slicing ensures smooth conversion of 3-hour sermons even on old office laptops without memory crashes.
* **KR**: 오디오 분할 기술을 통해 3시간 이상의 긴 설교도 구형 사무용 노트북에서 끊김이나 메모리 부족 없이 변환됩니다.

### 🚀 如何使用？ | How to Use | 사용 방법

您不需要懂任何代码，只需 3 步即可完成转换：
No coding skills required. Just 3 simple steps:
코딩 지식이 전혀 필요 없습니다. 다음 3단계만 따라 하세요:

**Step 1: 下载软件 | Download | 다운로드**
* **ZH**: 在本页面的 `Releases` 下载最新的 `FlowScribe_Lite_CPU.exe`。
* **EN**: Download the latest `FlowScribe_Lite_CPU.exe` from the `Releases` section on this page.
* **KR**: 이 페이지의 `Releases` 탭에서 최신 `FlowScribe_Lite_CPU.exe` 파일을 다운로드합니다.

**Step 2: 运行并配置 | Run & Set API | 실행 및 API 설정**
* **ZH**: 双击免安装打开。点击右上角【设置】，填入您的 DeepSeek API Key。
* **EN**: Double-click to open (no install needed). Click [Settings] to enter your DeepSeek API Key.
* **KR**: 설치 없이 더블 클릭하여 실행합니다. 우측 상단의 [설정]을 클릭하여 DeepSeek API Key를 입력합니다.

**Step 3: 开始转换 | Start Conversion | 변환 시작**
* **ZH**: 点击 **【➕ 添加本地音频】** (支持 mp3/wav/m4a)，然后点击左下角 **【▶️ 启动转换】**。完成后，点击右下角 **【📁 查看结果】** 即可直接打开文件夹查看完美讲章！
* **EN**: Click **[➕ Add Audio]**, select your file, and click **[▶️ Start]**. When finished, click **[📁 View Results]** to open the folder with your perfectly formatted text.
* **KR**: **[➕ 오디오 추가]** 를 클릭하여 파일을 선택하고, **[▶️ 변환 시작]** 을 누릅니다. 완료 후 **[📁 결과 보기]** 를 클릭하면 텍스트 파일이 있는 폴더가 자동으로 열립니다!

### 🛠️ 开发者指南 (For Tech Coworkers)

If you are a developer and want to run from source code or contribute:

**1. Clone & Install**
```bash
git clone https://github.com/raychen74771-lang/FlowScribe.git
cd FlowScribe
pip install -r requirements.txt
```
(Note: FFmpeg is required. Ensure it is added to your system environment variables.)

**2. Run Locally**
```bash 
python ui_main.py
```
**3. Contribute (Pull Requests Welcome!)**
* 🌍 Help translate the UI into more languages (Spanish, etc.).
* ⚡ Optimize underlying FFmpeg slicing logic.
* 📖 Improve documentation and create video demos.
